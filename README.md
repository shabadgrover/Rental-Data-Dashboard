# Rental Data Dashboard (Excel)

## 📌 Overview
This project focuses on building an interactive dashboard in Microsoft Excel to analyze rental data and uncover meaningful insights using visualization techniques.

## 🎯 Objectives
- Analyze rental trends and patterns
- Identify key metrics affecting rentals
- Present data in an interactive and visual format
- Enable quick decision-making through dashboards

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Conditional Formatting

## 📊 Key Features
- Interactive dashboard with filters and slicers
- Visual representation of rental trends
- Dynamic charts for better analysis
- Organized data for easy understanding

## 📈 Insights
- Identified patterns in rental trends
- Compared different categories and metrics
- Highlighted key factors influencing rentals

## 🚀 Outcome
This project demonstrates how Excel can be used as a powerful tool for data analysis and visualization through dashboards.

## 📌 Future Scope
- Automate data updates
- Integrate with Power BI for advanced analytics
- Expand dataset for deeper insights
